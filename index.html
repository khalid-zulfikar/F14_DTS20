Kami Merupakan Peserta Digital Talent Scholarship 2020 dari Digital Talent Scholarship yang diadakan oleh Kementerian Komunikasi dan Informatika RI. Kelompok Kami Mengikuti Pelatihan Tentang HTML, CSS, dan JavaScript dimana pada pelatihan tersebut Kementerian Komunikasi dan Informatika RI bermitra dengan Progate dan kami mendapatkan Kelas-Pro-F dan sekarang mendapat tugas ini dengan nama kelompok F14_DTS20.

Dalam membangun website ini kelompok kami menggunakan HTML dan CSS, Dimana untuk struktur website dibangun dengan 1 file index.html, dan untuk memperindah tampilan serta responsive pada beberapa perangkat kami menggunakan 2 file css yaitu stylesheet.css dan responsive.css.
